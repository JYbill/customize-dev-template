# koa2-template
## 开发环境
- nodejs v22(👍)
- npm/pnpm(👍)

## 目录结构规范
```
config            环境变量文件
src               项目源码
    enum          枚举常量
    library       第三方库统一配置入口
    middleware    中间件
    router        路由/controller
    service       服务/业务源码
    utils         工具
    index.js      入口文件
tmp               临时文件目录
```
